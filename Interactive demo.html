STDPFunction: The Hybrid Engine

How neuron.py separates Biology from Math

Standard Path (Blue): Calculates output = input @ weight and returns normal gradients (dL/dW) for the optimizer.
STDP Path (Orange): Calculates delta = trace_pre * spike_post and saves it to weight.stdp_delta. It does NOT touch gradients.
